    TKTween.tween(entity: Any, fn: Any) ¶ returns unknown

    Starts a potentially multi-step tween. Pass a function with no parameters that starts at least one TweenStep for this to do anything useful. entity - The entity that owns this tween. When the entity is destroyed, the tween will be destroyed. Note that the tween may operate on other entities besides the owner, but just be careful to cancel the tween before they are removed. 

        
// Multi step tween: will first fade in while rising up from 100 to 200 over 3 seconds, 
// then play a sound, then cycle through all the hues over 5 seconds.
TKTween.tween( _logo ) { 
  Log.print("Starting tween...")
  TweenStep.duration( 3.0 ).easing( Easing.cubic ).start { |tval| 
    Log.print("First tween step, tval is %(tval)")
    Sprite.set_alpha( _logo, tval )
    Transform.set_pos_y( _logo, Math.lerp( 100, 200, tval ) )
  }
  Audio.play("tada") // you can do side effects between tween steps!
  TweenStep.duration( 5.0 ).easing( Easing.cubic ).start { |tval| 
    Log.print("Second tween step, tval is %(tval)")
    var c = Color.color_from_hue(  tval )
    Sprite.set_color( _logo, c.r, c.g, c.b, 1.0 )
  }
  Log.print("Tween is finished.")
  Sprite.set_color( _logo, 1, 1, 1, 1  )
}

    TKTween.make_tween(entity: Any, duration: Any) ¶ returns unknown

    Makes a one-step function tween but doesn't start it yet. entity - The entity that owns this tween. duration - Duration of the tween in seconds. Returns a TweenStep, you can chain effects onto it and call start() on this with the tween function you want. 

    // Fades in _ent over 5 seconds
TKTween.make_tween( _ent, 5.0 ).easing( Easing.cubic ).start { |tval|
  Log.print("Tval is %(tval)")
  Sprite.set_opacity( _ent, tval )
}

    There is also a version of start that will take in two values and blend between them so you don't need to do the lerp yourself. This works on Sequences like vec2 and vec3 as well. 

    TKTween.make_tween( _logo, 1.0 ).repeatForever().start( [100, 100], pos ) { |p|      
  Transform.set_pos( _logo, p.x, p.y )
}

    TweenTransform¶

    import "tktween: modifier/tktween" for TweenTransform

    Helpers for the Transform modifier, so you don't need to manually tween commonly used effects for an entity.

    • move(ent: Any, duration: Any, pos_a: Any, pos_b: Any)
    • move_to(ent: Any, duration: Any, pos_dest: Any)
    • scale(ent: Any, duration: Any, scale_a: Any, scale_b: Any)
    • angle2D(ent: Any, duration: Any, degrees_a: Any, degrees_b: Any)
    • rotate_around_axis(ent: Any, duration: Any, axis: Any, degrees_a: Any, degrees_b: Any)

    TweenTransform.move(ent: Any, duration: Any, pos_a: Any, pos_b: Any) ¶ returns unknown

    Tweens from pos_a to pos_b over duration seconds. Equivalent to: 

    TKTween.make_tween( ent, duration ).start( pos_a, pos_b ) { |p|      
  Transform.set_pos( ent, p.x, p.y )
}
    Example: 
    TweenTransform.move( _logo, 1.3, [100, 500], pos )
              .repeat(2)
              .easing( Easing.cubic )
              .start()

    TweenTransform.move_to(ent: Any, duration: Any, pos_dest: Any) ¶ returns unknown

    Tweens from current position to pos_dest over duration seconds. Example: 

    TweenTransform.move_to( _logo, 1.3, pos )
              .repeat(2)
              .easing( Easing.cubic )
              .start()

    TweenTransform.scale(ent: Any, duration: Any, scale_a: Any, scale_b: Any) ¶ returns unknown

    Tweens scale beteen two values. Values given can be Nums for a uniform scale, or vecs for a non-uniform scale. Example: 

    // Make the logo appear with a bouncy pop-in effect
TweenTransform.scale( _logo, 2.0, 0.01, 1.0 )
              .easing( Easing.out_elastic )
              .start()
    Or non-uniform scale: 
    // Squash and stretch -- bounce from flat and wide to tall and skinny
TweenTransform.scale( _logo, 2.0, [2.0, 0.5], [0.5, 1.5] )
              .easing( Easing.inout_bounce )
              .start()

    TweenTransform.angle2D(ent: Any, duration: Any, degrees_a: Any, degrees_b: Any) ¶ returns unknown

    Tweens 2D rotation from angle_a to angle_b. Example: 

    TweenTransform.angle2D( _arrow, 3.0, 90.0, 270.0 )
              .easing( Easing.inout_bounce )
              .start()

    TweenTransform.rotate_around_axis(ent: Any, duration: Any, axis: Any, degrees_a: Any, degrees_b: Any) ¶ returns unknown

    Tweens rotation around axis from angle_a to angle_b Example: 

    // flip over
TweenTransform.rotate_around_axis( _logo, 1.4, [ 0, 1, 0], 0, 180 )
              .easing( Easing.inout_quartic )
              .start()
